# Backend

Carpeta reservada para la API del sistema.

## Sugerencia de modulos

- `auth`: autenticacion y control de acceso.
- `beneficiarios`: CRUD de beneficiarios.
- `colaboradores`: CRUD de personal operativo.
- `catalogos`: regiones, ciudades, cargos, tipos de ayuda y tipos de poblacion.
- `entregas`: registro y consulta de ayudas entregadas.
- `reportes`: consultas agregadas por ciudad, region, colaborador o periodo.

## Stack sugerido

- Node.js
- Express
- mysql2 o sequelize/prisma
- JWT
